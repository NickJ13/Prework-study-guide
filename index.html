HTML

  • HTML is one of the three cornerstone languages used in web development.
  • HTML is used to create the structure of a web page.
  • Made up of elements which work together to provide content and the basic framework for all pages on the web.

CSS

  • CSS: A programming language used by developers to define how webpage content is presented to users.
  • There are 3 ways to style a webpage using CSS: Inline CSS, Internal CSS style sheet, External CSS style sheet.
  • Inline CSS: when a developer wants to style an element directly in the HTML file.
  • Most common way to style a webpage is by using External CSS Style Sheet
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • ls command shows the list of files and directories.
  • CD command allows us to change directories.
  • pwd command will print out the directories.
  • MKDIR lets us create new files in directory.
  • Git status: Will show what branch we are on.
  • Git checkout -b branch-name: creates a new branch and switches to it.
  • Git pull: Used to fetch and download content from a remote repository and immediaty upddate the local repository to match that content.
  • Git push: Used to upload local repository content to a remote repository.
